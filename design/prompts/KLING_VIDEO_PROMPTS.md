# Kling 2.5 Video Prompt Templates for Prodowl

## Why Kling 2.5?

Kling 2.5 excels at:
- **Motion Quality**: Smooth, natural animations
- **Character Animation**: Good at animating characters
- **Longer Clips**: Up to 2 minutes of video
- **Human/Character Movement**: Excellent motion understanding

Use Kling for **mascot animations, UI micro-interactions, and marketing videos**.

---

## Video Generation Basics

### Kling Prompt Structure

```
[SUBJECT] + [ACTION] + [CAMERA MOVEMENT] + [ATMOSPHERE/STYLE]
```

### Technical Settings

```
RESOLUTION: 1080p or 4K
DURATION: 5-10 seconds (sweet spot)
ASPECT RATIO:
  - 16:9 (marketing, desktop)
  - 9:16 (mobile, social stories)
  - 1:1 (social posts)
FPS: 24 or 30
```

## Camera Movement Reference

### Static
```
Camera remains completely still
Use for: UI mockups, looping animations
```

### Push In
```
Camera slowly moves toward subject
Use for: Emotional moments, focus attention
```

### Pull Out
```
Camera moves away from subject
Use for: Reveals, context establishment
```

### Pan
```
Camera moves horizontally
Use for: Following action, transitions
```

### Tracking
```
Camera follows subject movement
Use for: Character in motion
```

### Slight Movement
```
Subtle camera drift or breathing
Use for: Adding life to static scenes
```

## Multi-Model Pipeline

### Kling + Midjourney + Runway

**Step 1**: Generate hero frame in Midjourney
```
Prodowl owl mascot [pose], Pixar style, white background
```

**Step 2**: Animate in Kling
```
[Upload Midjourney image]
Animate this character [action], smooth motion
```

**Step 3**: Enhance in Runway (optional)
```
[Upload Kling output]
Add camera movement: slow push in
Enhance: Add subtle glow effects
```

**Step 4**: Post-production
- Color grade for consistency
- Add sound effects
- Export at target resolution
