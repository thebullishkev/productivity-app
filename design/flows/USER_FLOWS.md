## Deep Linking Architecture

### Internal Deep Links (prodowl://)

| Route | Parameters | Action |
|-------|------------|--------|
| `prodowl://tasks` | `filter`, `priority`, `category` | Open tasks view with filters |
| `prodowl://tasks/{id}` | - | Open specific task |
| `prodowl://habits` | - | Open habits view |
| `prodowl://habits/{id}` | - | Open specific habit |
| `prodowl://timer` | `task_id`, `duration` | Start timer for task |
| `prodowl://notes` | - | Open notes view |
| `prodowl://notes/{id}` | - | Open specific note |
| `prodowl://web3` | `chain`, `action` | Open Web3 view |
| `prodowl://social` | `platform` | Open social tasks |
| `prodowl://settings` | `section` | Open settings |

## Notification Trigger Points

### Time-Based Triggers

| Trigger | Condition | Message Type | Mascot Mood |
|---------|-----------|--------------|-------------|
| Morning Check-in | 9:00 AM | Encouraging | Happy |
| Afternoon Nudge | 2:00 PM, <50% tasks | Passive-Aggressive | Scheming |
| Evening Reminder | 6:00 PM, incomplete | Guilt | Sad |
| Streak Warning | 10:00 PM, no completions | Urgent | Angry |

### Event-Based Triggers

| Event | Response | Message Type |
|-------|----------|--------------|
| Task Completed | Celebration | Happy |
| Streak Achieved | Major Celebration | Celebrating |
| Streak Broken | Disappointment | Sad |
| Deadline Approaching | Urgency | Urgent |
| Long Absence | Welcome Back | Passive-Aggressive |
| Multiple Tasks Done | Encouragement | Happy |
